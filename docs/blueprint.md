# **App Name**: Balatasan Resort Hub

## Core Features:

- User Authentication: Secure user registration and login system with different roles (guest, staff, admin) and JWT-based authentication.
- Accommodation Management: Browse and book accommodations, view availability, and manage bookings.
- Tour Package Booking: Browse and book tour packages, view availability, and manage bookings.
- Secure Online Payments: Integrate with PayMongo SDK (GCash, PayMaya) for secure online payments.
- Booking Management: Manage bookings, update status, and generate booking references.
- Review and Rating System: Users can submit reviews and ratings for accommodations and tours.  An LLM tool can be used to screen for inappropriate content, and suggest edits. 
- Admin Dashboard: A comprehensive dashboard for managing users, accommodations, tours, bookings, and payments.

## Style Guidelines:

- Primary color: Ocean Blue (#3498db) to evoke a sense of tranquility and seaside experience.
- Background color: Light Beige (#f5f5dc) to create a calm and inviting atmosphere.
- Accent color: Coral Orange (#e74c3c) for highlighting key actions and elements.
- Body font: 'PT Sans' sans-serif for readability and a modern, clean interface
- Headline font: 'Playfair' serif for elegance
- Use clean, outlined icons representing accommodations, tours, and activities.
- Subtle transitions and animations for a smooth user experience, such as fading effects and sliding panels.
